# QS Assets

Internal CDN + asset manager for QuietStates.

## Features
- Upload files
- Categorize assets (audio, thumbnails, logos, etc.)
- View & download
- Discord webhook integration

## Setup

### Install all deps
npm install
cd backend && npm install
cd ../frontend && npm install

### Run dev
npm run dev

Backend: http://localhost:5000  
Frontend: http://localhost:5173
